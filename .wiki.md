Single-page brochure website for Таисия Галицкая, a psychologist working online (psychoanalytic approach, adults only).

## What it does

A calm, no-pressure landing page: hero, what-to-come-with, how-I-work, about, education, pricing, FAQ, and contacts. Contact is via Telegram, email, and VK only — no phone, no form, no analytics — so the site collects zero personal data and stays out of 152-ФЗ scope. Plain HTML/CSS with a little vanilla JS for scroll-reveal and the mobile menu; `build-single.py` inlines everything into one shareable file. Editorial-calm design — cream + terracotta palette, Lora + Manrope type.

## Key decisions

- Custom plain HTML/CSS over Tilda or a framework — chosen for zero-maintenance longevity on a rarely-edited single page (supersedes the Tilda default in `RESEARCH.md`)
- Hosted as static files on Yandex Object Storage (RF), HTTPS via Certificate Manager + Cloud CDN
- No phone / no form / no Metrika — keeps the site free of personal-data collection
- Copy written by the psychologist herself; treated as verbatim
