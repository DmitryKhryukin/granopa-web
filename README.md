# Сайт Таисии Галицкой — психолог

Одностраничный сайт-визитка психолога: спокойная подача, контакты через
мессенджеры, без формы и сбора данных. Чистый HTML/CSS — открыл файл, и это
уже готовый сайт.

---

## ✨ Что это

- **Одна страница** со всеми блоками: первый экран → с чем прийти → как работаю →
  обо мне → образование → стоимость → как начать + FAQ → контакты
- **Без сборки и зависимостей** — чистый HTML/CSS + немного ванильного JS
- **Без формы, телефона и аналитики** — сайт не собирает персональных данных
  (осознанно, чтобы не попадать под 152-ФЗ)
- **Тёплый «бумажный» дизайн** — палитра крем + терракота, шрифты Lora + Manrope
- **Готов к хостингу на Yandex Object Storage** (статика, РФ)

## 🚀 Быстрый старт

### Требования

- Любой современный браузер — чтобы посмотреть сайт
- **Python 3** — только для двух необязательных удобств (локальный сервер и
  сборка одного файла)

### Просмотр

Самое простое — открыть `index.html` двойным кликом.

Либо поднять локальный сервер (тогда корректно работают относительные пути):

```sh
python3 -m http.server 8000
# открыть http://localhost:8000
```

## 📖 Использование

### Отправить страницу на просмотр (например, жене)

Сайт — это три файла (`index.html` + `styles.css` + `main.js`), которые работают
вместе. Чтобы отправить **один файл**, собери самодостаточную версию:

```sh
python3 build-single.py
# → таисия-сайт.html  (стили и скрипт вшиты внутрь)
```

Полученный `таисия-сайт.html` можно переслать как обычный файл — он открывается
двойным кликом и показывает всё оформление. После любых правок пересобери заново.

### Что заполнить перед публикацией

Открой `index.html`, найди (Ctrl/Cmd+F) слово **`ЗАПОЛНИТЬ`** — там стоят заглушки:

| # | Что | Заглушка |
|---|---|---|
| 1 | Телеграм для записи | `https://t.me/USERNAME` |
| 2 | Почта | `mailto:EMAIL` |
| 3 | ВКонтакте | `https://vk.com/USERNAME` |
| 4 | Канал «Достаточно хороший» | `https://t.me/CHANNEL` |
| 5 | Дзен | `https://dzen.ru/USERNAME` |
| 6 | og:url и og:image | в `<head>` |

### Фото

Положи картинки в папку `assets/` и замени блоки-заглушки в HTML. Было:

```html
<div class="photo-ph photo-ph--portrait"> … </div>
```

Стало:

```html
<img src="assets/hero.jpg" alt="Таисия Галицкая" />
```

Нужны: `assets/hero.jpg` (первый экран), `assets/about.jpg` (блок «Обо мне»),
`assets/diploma-1.jpg`, `diploma-2.jpg`, … (дипломы).

## ⚙️ Шрифты

Сейчас Lora + Manrope грузятся с Google Fonts — для быстрого старта. Перед
публикацией их стоит **самохостить**, чтобы убрать внешний запрос и не зависеть
от доступности Google:

<details>
<summary>Как самохостить шрифты</summary>

1. Скачать woff2 (например через [google-webfonts-helper](https://gwfh.mranftl.com)).
2. Положить в `assets/fonts/`.
3. Добавить `@font-face` в `styles.css`, убрать `<link>` на Google Fonts из `index.html`.

</details>

## 🛠️ Деплой на Yandex Object Storage

1. Создать бакет, включить **хостинг статических сайтов**, главную — `index.html`.
2. Загрузить `index.html`, `styles.css`, `main.js` и папку `assets/`.
3. Для своего домена и HTTPS — выпустить бесплатный сертификат в **Certificate
   Manager** и поставить **Cloud CDN** перед бакетом.

Подробности — [документация Яндекса](https://cloud.yandex.ru/docs/storage/concepts/hosting).
